###### 书源提升权重很简单，进入书源管理，左上角排序用来调整想要改变的权重，右上角菜单栏，点击编辑模式，这时候按住每个书源的右端可以拖动排序，书源排列的顺序就是对应排序下的权重
###### 暂时禁用可以到设置，书源，把第一项设置为不禁用即可
